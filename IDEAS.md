# Mova - Feature Ideas

## Learn New Words

### Pronunciation & Feedback

- [x] **Fuzzy pronunciation matching** — Levenshtein/phoneme similarity instead of exact string match; reduce false negatives from accent variation
- [x] **Pronunciation score instead of pass/fail** — Show 0–100% similarity so kids see improvement even when not perfect
- [x] **Slow/fast audio speed select** — 0.5×, 0.75×, 1×, 1.25× dropdown in learn + review sessions
- [x] **Syllable highlighting** — Show word broken into syllables as audio plays

### Learning Mechanics

- [ ] **Pre-teach with context** — Show a simple sentence using the word before asking for pronunciation
- [ ] **Multiple choice mode** — 4 English options to match the foreign word before attempting pronunciation; good for early learners
- [ ] **Harder/easier in-session routing** — Track per-word failure rates and re-insert struggling words into same session instead of waiting for SM-2
- [ ] **Category grouping** — Let users choose to learn by category (animals, food, etc.) instead of pure frequency order
- [ ] **Mini-quiz at session end** — After 10 words, do a 3-question recap to reinforce before closing

### Image & Audio

- [ ] **Contextual scene images** — Show word used in a simple scene instead of isolated object on white background
- [x] **Native speaker / child voice option** — Offer a second TTS voice (e.g. child's voice) alongside current Rachel voice
- [ ] **Prominent audio replay button** — Make speaker button more visible during recording so kids can re-hear before attempting

### Spaced Repetition Improvements

- [ ] **Difficulty self-rating** — After each word, kid rates "Easy / Hard" to calibrate SM-2 ease factor (currently only implicit from pass/fail)
- [ ] **Retire mastered words** — If a word is reviewed correctly 5+ times in a row, mark it retired and remove from review queue
- [ ] **Same-day re-review for failures** — If a word fails during review, add it back into the same session (not just interval: 0 for next day)

### Engagement & Motivation

- [ ] **Word of the day** — Surface one featured word on the dashboard with an interesting fact or image
- [ ] **Parent progress report** — Weekly summary with which words were learned and which need more practice
- [ ] **Bonus words on level completion** — Unlock culturally interesting words (holidays, animals, food) when a level is completed
- [ ] **Category-themed sound effects** — Animal words get animal sounds, food words get eating sounds, etc.
- [ ] **Personalized word order** — Deprioritize categories where a child consistently struggles in favor of more wins

### UX & Accessibility

- [ ] **Keyboard shortcut to replay audio** — Space bar replays the word between recording attempts
- [ ] **Progress preview before session** — Show which 10 words are coming up before starting so kids feel prepared
- [ ] **Session length control** — Let users choose 5, 10, or 15 words per session (currently fixed at 10)
- [ ] **Offline mode** — Cache pre-generated audio/images so app works without internet for already-seen words
